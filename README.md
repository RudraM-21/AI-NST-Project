# ai-nst-project
📌 Overview

AI Sketch Studio is an AI-powered image transformation application that converts normal images into artistic pencil sketches, anime sketches, charcoal art, and stylized line drawings.

The project uses computer vision and deep learning models to process uploaded images and generate high-quality sketch transformations in real time.

Users can upload photos and instantly apply multiple sketch styles with adjustable intensity and artistic effects.

✨ Features
🖼️ Upload and transform images instantly
✏️ Convert photos into pencil sketches
🎨 Multiple artistic sketch styles
⚡ Real-time image processing
🤖 AI-powered image transformation
🌈 Adjustable sketch intensity & contrast
📱 Responsive modern UI
☁️ Cloud deployment support
💾 Download transformed images
🔥 Fast inference pipeline
